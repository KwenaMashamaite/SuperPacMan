# Super Pac-Man

[Super Pac-Man](https://en.wikipedia.org/wiki/Super_Pac-Man) is an arcade game that was created in 1982 as a spin-off from the popular Pac-Man game. The aim of this project is to create a Super Pac-Man clone using modern C++, the [Infinite Motion Engine](https://github.com/KwenaMashamaite/IME) and the doctest unit testing framework.
