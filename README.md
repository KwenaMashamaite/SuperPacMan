# Super Pac-Man

[Super Pac-Man](https://en.wikipedia.org/wiki/Super_Pac-Man) is an arcade game that was created in 1982 as a spin-off from the popular Pac-Man game. This project is a Super Pac-Man clone using C++17). In addition, the project is a demo for the [Infinite Motion Engine](https://github.com/KwenaMashamaite/IME), a 2D game engine built on top of SFML. The engine is more suited for tile based games and Super PacMan is the perfect candidate.

The core aim of the project is to illustrate extensive use of the engines features while building a classic game (and having fun :) ). I would greatly appreciate it if you could check out the engine (browse the source code or use the engine in one of your projects) and leave some feedback or suggest new features etc...

## How to play

1. Download the [game]()
2. Extract the zip archive
3. Run ``SuperPacMan.exe``
4. Enjoy :)

## Platform

This game is developed for the Windows platform. It was tested on the latest version of windows 10, but should work with other versions from Windows 7.
